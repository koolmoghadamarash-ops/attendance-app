# ⚽ Futsal Coach Pro

اپ مربیگری فوتسال با گرافیک حرفه‌ای، انیمیشن و قابلیت ذخیره و خروجی

## امکانات

- **زمین فوتسال واقعی** با گرافیک SVG و جزئیات کامل
- **۱۰ بازیکن** (۵ در برابر ۵) قابل جابجایی با لمس
- **توپ** قابل جابجایی روی زمین
- **ابزار نقاشی:**
  - قلم آزاد (حرکات نرم)
  - فلش جهت‌دار
  - خط مستقیم
  - منطقه (دایره نقطه‌چین)
  - پاک‌کن
- **رنگ‌ها:** ۸ رنگ + ۳ ضخامت خط
- **انیمیشن فریم به فریم** با timeline
- **ذخیره** بازی‌ها با AsyncStorage
- **خروجی** PNG به گالری یا اشتراک‌گذاری

## نصب و راه‌اندازی

### پیش‌نیازها
- Node.js 18+
- Expo CLI: `npm install -g expo-cli`
- اپ Expo Go روی گوشی (از App Store / Google Play)

### اجرا

```bash
cd futsal-coach
npm install
npx expo start
```

سپس QR کد را با اپ Expo Go اسکن کنید.

### برای Android/iOS (build کامل)

```bash
npx expo run:android
# یا
npx expo run:ios
```

## نحوه استفاده

### طراحی تاکتیک
1. **ابزار انتخاب (☝️):** بازیکنان و توپ را بکشید
2. **ابزار قلم (✏️):** مسیر حرکت رسم کنید
3. **ابزار فلش (➡️):** جهت پاس/حرکت نشان دهید
4. **ابزار خط (📏):** خط مستقیم
5. **منطقه (⭕):** ناحیه را مشخص کنید

### انیمیشن
1. بازیکنان را در موقعیت اول قرار دهید → **+ فریم**
2. بازیکنان را به موقعیت بعدی ببرید → **+ فریم**
3. تکرار کنید...
4. دکمه **▶ پخش** برای نمایش انیمیشن

### ذخیره و خروجی
- **💾 ذخیره:** تاکتیک با نام دلخواه ذخیره می‌شود
- **📤 خروجی:** عکس PNG در گالری ذخیره یا share می‌شود
